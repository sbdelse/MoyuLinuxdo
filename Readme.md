# 本项目是一个用于在 Linux.do 论坛的命令行工具，用于在命令行摸鱼阅读帖子。
- 使用 C# 编写
- 使用 .NET 8.0 SDK 编译
- 使用方法：
    - 直接运行编译后的二进制文件，需要安装 .NET 8.0 SDK
    - 或者 git clone 本项目后使用 `dotnet run` 运行
- 设置 cookie 理论上只需要设置 `_t` 即可。可以浏览器 F12 后在 `Application` 的 `Cookies` 里面找到，或者在 `Network` 的 `Cookie` 里面找到。
